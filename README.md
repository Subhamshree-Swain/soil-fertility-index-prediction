# Soil Fertility Index Prediction and Mapping

This project predicts the soil fertility index using the soil parameters.
Knowing the SFI values at some points in the field, this model interpolates for the rest of the field points.
Random Forest Model is used train this model.

---

## Installation


```bash
git clone https://github.com/Subhamshree-Swain/soil-fertility-index-prediction.git
cd soil-fertility-index-prediction
pip install -r requirements.txt
```
---

##  Running the project

```bash
python3 src/main.py
```
